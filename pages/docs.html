Getting Started

Learn the basics and get up and running quickly.

Installation

Set up Viseyyon CLI and connect to your account.

# Install via npm
npm install -g @viseyyon/cli
# Login to your account
viseyyon login

Configuration

Configure your project settings and API keys.

# Initialize project
viseyyon init
# Set API key
viseyyon config set api-key YOUR_KEY
